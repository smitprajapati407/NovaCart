# NovaCart 🛒

NovaCart is a modern e-commerce web application built with Django.

It includes a complete customer shopping experience along with a custom admin dashboard for managing products, categories, users, and orders.

---

## Features

### Customer Features

* User Registration & Login
* Product Browsing
* Categories & SubCategories
* Product Details Page
* Search Products
* Wishlist
* Shopping Cart
* Checkout System
* Order Placement
* Order History
* Product Reviews & Ratings

### Admin Dashboard

* Dashboard Analytics
* Revenue Tracking
* Product Management
* Category Management
* SubCategory Management
* User Management
* Order Management
* Order Status Updates
* Recent Orders Overview

## Installation

Clone the repository:

```bash
git clone https://github.com/smitprajapati407/NovaCart.git
```

Navigate into project:

```bash
cd shopify
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run migrations:

```bash
python manage.py migrate
```

Start server:

```bash
python manage.py runserver
```
